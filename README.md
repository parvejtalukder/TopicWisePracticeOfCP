# TopicWisePracticeOfCP

A comprehensive collection of **competitive programming problems**, organized by topic, curated and maintained by **Parvej Husen Talukder**. This repository serves as a personal practice log and a resource for anyone looking to improve their problem-solving skills in **C++**.

## 📂 Repository Structure

The repository is organized **topic-wise**, with each folder containing problems and solutions:

- **AdhocMadness/** – Ad-hoc problem solving practice  
- **Array/** – Array-related problems and implementations  
- **Bitwise/** – Bit manipulation problems  
- **Core/** – Fundamental concepts and practice  
- **DataStructures/** – Problems related to various data structures  
- **FixIt/** – Debugging and problem-fixing exercises  
- **GCD/** – Greatest Common Divisor practice problems  
- **Greedy/** – Greedy algorithm problems  
- **LCM/** – Least Common Multiple problems  
- **LambdaFunc/** – Practice with lambda functions  
- **Log/Prac/** – Logging and miscellaneous practice problems  
- **Modulo/** – Problems involving modular arithmetic  
- **PRAC/** – General practice problems  
- **PrecendenceandAssociativity/** – Operator precedence and associativity exercises  
- **SearchA/** – Search algorithms and problems  
- **Sortign/** – Sorting algorithms and exercises  
- **String/** – String manipulation problems  
- **WhiteSpace/** – Problems focusing on input/output and formatting  
- **GreedyProblemAnother/** – Additional greedy algorithm practice  

## 💻 Languages Used

- **C++** – 99.9% of problems
